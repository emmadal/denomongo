# DENO Rest API

> This is a DENO REST API written in Typescript. We're using Oak and MongoDB

## Get started

```sh
1. git clone https://github.com/emmadal/denomongo
2. cd denomongo
3. deno run --allow-net --allow-read --allow-write --allow-plugin --unstable server.ts
```

### API Routes

```js
GET /employees
GET /employee/:id
POST /employee
DELETE /employee/:id
PUT /employee/:id
```
